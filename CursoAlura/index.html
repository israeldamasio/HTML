Curso HTML & CSS - ALURA

Este é um curso de Front-end, que visa ensinar ao DEV como criar sites utilizando a ferramenta HTML(Hypertext Marcation Language) e a CSS(Cascading Style Sheets).

Cada uma dessas linguagens de programação tem a sua especificidade, funcionalidade e sentido na estruturação do conteúdo.

HTML - Função de marcar dentro do conteúdo a ser publicado qual a organização semântica entre os elementos de texto.

CSS - Função de dar estilo ao conteúdo marcado pelo HTML, de acordo com a intenção desejada pelo DEV.

  • Linguagem
  • Tag's
  • Plataformas
